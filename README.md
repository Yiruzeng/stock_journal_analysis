📊 策略回測系統 (Stock Journal Analysis)
這是一個專為投資者打造的股票買賣紀錄日誌系統，結合量化數據追蹤與 AI 深度覆盤，提供客觀的交易檢討方向與策略優化建議。

🚀 20260319 版本更新 V1.0: 現代化理財儀表板 (Dashboard Edition)
🎨 全新 UI 視覺重構 (Modern Dashboard UI)：
全面覆寫 Streamlit 原生樣式，導入現代化 SaaS 儀表板設計語彙。採用淺灰白背景搭配白色圓角卡片 (Card UI) 與微陰影，並以溫暖的磚紅色作為主視覺強調色。首頁 (Landing Page) 引入精美的置中插圖與歡迎標語，大幅提升系統專業度與操作體驗。

📅 彈性日曆區間篩選：
全面升級過濾機制，側邊欄新增「時間區間」日曆選擇器。不再受限於單一月份，使用者可自由框選特定日期範圍，系統將精準結算該區間內的投資績效、已實現損益與庫存狀態。

📊 總體績效儀表板與持倉總表：
主畫面頂部新增四大核心數據卡片（總投入金額、當前市值、總報酬率、已實現損益），並加入「當前庫存部位」數據總表，讓你一眼掌握資金水位、個別成本與未實現損益。

📑 個股分頁與狀態燈號 (Tabbed Interface)：
股票細節檢討改為活頁簿 (Tabs) 介面，分頁標題自動解析顯示「股票代號 + 中文名稱」。

⚡ 延遲運算與 API 成本優化 (On-Demand Analysis)：
本次核心架構升級！ 進入個股分頁時，系統僅會預先載入輕量級的「進出場明細表」與「K線圖」。最耗費網路資源與 API 額度（如 Gemini / OpenAI、FMP 基本面數據）的深度覆盤，改為「點擊按鈕才觸發執行」。並導入 Session State 快取技術，切換分頁報告不會消失，有效管控一人工作室的營運與 API 呼叫成本。

📈 專業級 K 線標註視覺化：
K 線圖樣式全面白底化以融入卡片 UI。圖表會自動計算並繪製 MA5 / MA20，同時將使用者的「買入(▲)」與「賣出(▼)」動作直接標註於歷史 K 線圖的對應價位上，滑鼠懸停即可顯示「交易理由」與「當次損益」，讓覆盤更加直觀。

📥 內建標準範本一鍵下載：
防呆機制升級！左側選單新增「CSV 記帳範本」下載按鈕，內含台股 (如 2330.TW) 與美股 (如 NVDA) 的標準填寫範例。採用 UTF-8-SIG 編碼，確保使用者下載後直接用 Excel 開啟，中文也不會出現亂碼。
